# Advent of Code 2023

A repository to house solutions from Advent of Code 2023, all solutions are written in Python. The goal is to sharpen up my knowledge of Python and to follow latest changes of the language.

## Summary

[Advent of Code](http://adventofcode.com/) is an Advent calendar of small programming puzzles for a variety of skill sets and skill levels that can be solved in any programming language you like. People use them as interview prep, company training, university coursework, practice problems, a speed contest, or to challenge each other.
